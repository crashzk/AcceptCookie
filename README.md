# AcceptCookie

Модуль для вывода информации о подтверждении Cookie. Встраивается автоматически в шаблон и не показывается после подтверждения.

Module for displaying information about the confirmation of Cookie. Automatically inserted into the template and does not show when confirming.

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **2.0.0**
